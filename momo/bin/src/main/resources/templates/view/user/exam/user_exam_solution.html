비타협적 민족주의자와 사회주의자의 민족 협동 전선으로 조직된 (가)이/가 창립총회를 개최하여 이상재를 회장으로 선출하였다. 이 단체는 지난달 다음과 같은 3대 강령을 발표하였다.

- 우리는 정치적ㆍ경제적 각성을 촉진함.

- 우리는 단결을 공고히 함.

- 우리는 기회주의를 일체 부인함.

  1. ①  가갸날을 제정하였다.
  2. ②  진단 학보를 발행하였다.
  3. ③  교조 신원 운동을 전개하였다.
  4. ④  광주 학생 항일 운동을 지원하였다.
  5. ⑤  일제의 황무지 개간권 요구를 철회시켰다.

  4. ④ 병자호란의 전개 과정을 알아본다.
